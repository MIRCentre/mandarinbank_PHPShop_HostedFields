==================================================
============= ИСТРУКЦИЯ ПО УСТАНОВКЕ =============
==================================================

1. Распаковываем содержимое репозитория в корень сайта;
2. Авторизуемся в админ панели(/phpshop/admpanel);
3. Переходим в Модули > Управление модулями(/phpshop/admpanel/admin.php?path=modules);
4. Переходим в Продажи(/phpshop/admpanel/admin.php?path=modules&cat=sale);
5. Находим модуль "MandarinPay Hosted";
6. Нажимаем на шестерёнку > Включить;
7. Нажимаем на шестерёнку > Настройки(/phpshop/admpanel/admin.php?path=modules&id=mandarinhosted);
8. Вкладка Инструкции: ознакамливаемся, как узнать свой id и секретный ключ;
9. Вкладка Основное: Заполняем ID Мерчанта, Секретный ключ;
10. Нажимаем Сохранить;
11. Переходим в Заказы > Способы оплаты(/phpshop/admpanel/admin.php?path=payment);
12. Находим и нажимаем на MandarinPay(/phpshop/admpanel/admin.php?path=payment&id=10027);
13. Группа полей "Сообщение после заказа": заполняем Заголовок, Сообщение - эти данные будут отображаться покупателю в случае удачной оплаты;
14. Нажимаем Сохранить и закрыть;

Теперь платежная система MandarinPay через API Hosted Fields настроена на вашем сайте!
Успешных платежей!
